# Ateler Workshop

## Introduction à la scalabilité backend

Atelier Workshop est une application de gestion d'ateliers qui permet aux utilisateurs d'acheter des billets pour des ateliers.

### Installation

1. Cloner le dépôt

```bash
git clone https://github.com/Dev2901/workshop-v0.git
cd workshop-v0
npm install
```

## Démarrage

```bash
docker-compose up --build
```
